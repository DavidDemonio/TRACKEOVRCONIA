Coloca aquí los modelos ONNX y los assets de MediaPipe necesarios para ejecutar el tracking sin depender de los CDN.

- Consulta el documento `../../../../docs/MODEL_ASSETS.md` para obtener la lista de URLs oficiales y la ruta exacta de cada archivo.
- Si no copias nada en esta carpeta, la aplicación descargará los recursos directamente desde los CDN de MediaPipe y ONNX en tiempo de ejecución.
- Recuerda eliminar los binarios antes de publicar cambios en Git. Sólo deben quedar los archivos de texto (`.gitignore`, este README).
